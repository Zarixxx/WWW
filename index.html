// 02 · PROJECT

ZAPD

v1.1.0 · C · LINUX
ZapD
// Network & Threat Analysis Tool

Herramienta de análisis de redes y amenazas escrita en C. Binario nativo sin dependencias externas — compila una vez, ejecuta en cualquier Linux. Inspirada en nmap y VirusTotal.

⬡
PORT SCANNER
TCP/UDP real con hilos, detección de banners, OS fingerprint y control de timing T1-T5.
◈
VIRUSTOTAL
Integración con API v3. Analiza URLs, IPs, dominios y hashes contra 70+ motores AV.
⌖
WHOIS / DNS
Consulta directa a servidores WHOIS por TCP. Registros A, AAAA, MX, NS, TXT.
▸
PING / TRACE
Ping ICMP con estadísticas RTT y traceroute hop-by-hop con resolución de hostnames.
